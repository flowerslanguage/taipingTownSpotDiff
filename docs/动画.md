Cocos Creator 动图/动画完整制作规范（项目自用版）
一、Cocos 中两种动画形式区别
Cocos 不支持直接播放 GIF 动图，项目中所有动态效果只分两类：

1. 序列帧动画（原本的 GIF 动图效果）：多张图片快速切换实现动态（火焰、光斑、小图标动效）
2. 属性动画（编辑器关键帧动画）：位置、缩放、旋转、透明度变化（云朵飘动、河水流淌、太阳旋转）

---

二、GIF 动图转序列帧动画（标准流程）
适用：原本是 gif 动图的素材，Cocos 无法直接使用，必须拆成序列帧图片。

1. 素材处理
   将 GIF 拆解为一张张 png 序列帧，整理为一个文件夹，整体放入项目图片资源目录中。
2. 场景节点搭建
3. 将序列帧 第一张图片拖入场景，生成 Sprite 节点
4. 调整好尺寸、位置
5. Sprite 组件 Size Mode 改为 CUSTOM
6. 取消 Trim（去白边），防止帧与帧之间尺寸跳动抖动
7. 添加动画组件
8. 给节点添加 Animation 组件
9. 可提前勾选 Play On Load（加载自动播放）
10. 新建动画剪辑并绑定
11. 资源管理器新建 Animation Clip 动画文件
12. 打开底部动画编辑器，将动画剪辑拖入编辑器
13. 选中场景中的 Sprite 节点，自动绑定轨道节点
14. 生成序列帧关键帧（核心步骤）
15. 动画编辑器中，添加轨道：cc.Sprite → spriteFrame
16. 设置 帧间隔（常用 4～8，数值越大动画越慢）
17. 资源管理器 Ctrl+A 全选所有序列帧图片
18. 直接拖入轨道中，引擎 自动批量生成全部关键帧
19. 最终设置
20. 左下角播放模式选择 循环播放（按需选择单次/循环）
21. 顶部预览按钮播放检查流畅度
22. Ctrl+S 保存动画剪辑
    帧间隔数值说明：
    帧间隔=每帧停留帧数，数值越小越快，默认6为中等速度，根据动图快慢微调即可。

---

三、属性关键帧动画（位移/缩放/旋转/透明）
适用：云朵飘动、河水滚动、物体平移、缩放呼吸、旋转、淡入淡出等 持续运动效果。

1. 制作逻辑（类似 CSS 动画）
   通过记录「起点关键帧」和「终点关键帧」，引擎自动补中间动画。
2. 标准步骤
3. 新建/选中需要动画的节点
4. 新建动画剪辑，拖入动画编辑器
5. 添加对应轨道：

- 移动：UITransform → position
- 缩放：UITransform → scale
- 旋转：UITransform → rotation
- 透明：UITransform → opacity

4. 设置动画曲线：匀速运动统一用 Linear
5. 0帧记录初始状态、末尾帧修改目标状态
6. 开启循环、保存预览

---

四、多节点共用同一个动画（核心高效技巧）
适用：多朵云、多片雪花、多条水流需要速度、方向完全一致

1. 创建一个 空父节点（无需贴图、无需宽高）
2. 把所有需要同步运动的子节点放入父节点下
3. 只给父节点做动画
4. 所有子节点跟随父节点整体同步运动，保留各自初始位置差异
   优点：只需要 1 份动画资源，性能极高、不会卡顿、统一速度、零抖动。

---

五、核心总结（最简记忆）

- 动图效果（GIF） = 序列帧切换动画 → 靠切换图片实现
- 运动效果（飘动/滚动） = 属性关键帧动画 → 靠改位置、旋转、缩放实现
- 多物体同步运动 = 空父节点统一带动画，子节点只摆位置不动动画
